## Run Locally

Clone the project

```bash
  git clone https://github.com/Mathieu-ai/PNPM-Structure-Project
```

Go to the project directory

```bash
  cd PNPM-Structure-Project
```

Install dependencies

```js
  // use --filter APP_NAME to install only in the desired project or -r
  pnpm i
```

Start the server

```js
  // use --filter APP_NAME to install only in the desired project or -r
  pnpm run dev
```
